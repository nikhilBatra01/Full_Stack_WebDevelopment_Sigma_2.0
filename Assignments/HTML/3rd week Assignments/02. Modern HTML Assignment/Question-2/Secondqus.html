What are HTML entites.

An HTML entity is a piece of text ("string") that begins with an ampersand (&) and ends with a semicolon (;). Entities are frequently used to display reserved characters (which would otherwise be interpreted as HTML code), and invisible characters (like non-breaking spaces). You can also use them in place of other characters that are difficult to type with a standard keyboard.

List of 5 commonly Used HTML entites.

  1. "<" => Interpreted as the beginning of a tag
  2. "©" => Interpreted as the copyright sign.
  3. "£" => Interpreted as the pound symbol.
  4. "≈" => Interpreted as almost equal to sign.
  5. "–" => Interpreted as the en dash (half the width of an em unit).
