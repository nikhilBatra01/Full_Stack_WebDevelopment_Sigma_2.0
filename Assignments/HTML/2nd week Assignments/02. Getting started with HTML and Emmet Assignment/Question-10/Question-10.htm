Global attributes

In HTML, global attributes are attributes that can be used on any HTML element, regardless of its type. These attributes provide common functionality that is useful for many different types of elements. Here are 5 examples of global attributes:

  1. "class" - Used to assign one or more CSS classes to an element for styling and scripting purposes.
  2. "id" - Used to uniquely identify an element within a document.
  3. "style" - Used to apply inline CSS styles to an element.
  4. "title" - Used to provide additional information about an element (often displayed as a tooltip).
  5. "data" - Used to store custom data attributes on an element, which can be accessed via JavaScript.
