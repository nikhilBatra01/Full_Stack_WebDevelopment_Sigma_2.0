1. (b) tag or (strong) tag
    • Render text in bold. Ex:This is just text.
  2. (i) tag or (em) tag
    • Render text in italics. Ex: This text just italics.
  3. (sup) tag
    • Render text as superscript(rasied above the baseline). Ex: E=mc2.
  4. (sub) tag
    • Render text as subscripted(lowered below the baseline). Ex: H2o.
  5. (mark) tag
    • Highlight text with a background color. Ex: This text is highlighted.
  6. (u) tag
    • Render text with a underline. Ex: this text is Underlined.
  7. (s) tag or (del) tag
    • Renders text with a strikethrough (line through the text). Ex: This text has a strikethrough.
  8. (h1) to (h6) tag
    • These tags are used to define headings of different levels, with

      being the highest level and

      being the lowest level. Example:

      This is a heading level 1

  9. (Pre) tag
    • Preserves the formatting and whitespace of the text exactly as it appears in the HTML code, useful for displaying code snippets or preformatted text. Example: 
      def hello(): print("Hello, world!")
  10. (a) tag
    • Creates a hyperlink to another web page or resource. Example: Click here
