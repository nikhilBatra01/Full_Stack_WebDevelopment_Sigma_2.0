1. HTML TAG

    Html stand for Hyper text markup language. Is the basic scripted language used by web browser to render pages on the world wide web.

  2. A TAG

    A tag create a hyperlink.

  3. H1 TO H6 TAG

    This tag create a headings.

  4. IMG TAG

    This tag create an image, and this tag is not self closing tag.

  5. U TAG

    U tag create a underline.

  6. P TAG

    P tag create a paragraph.

  7. Head Tag

    Head tag is bassically used for styling on the web page and show the websites name.

  8. Body Tag

    The body tag defines the document's body. The body element contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

  9. Table Tag

    Table tag create a table. In this table,we can also used some property name [TR] element define a table row, [TD] element define a table cell.

  10. Ordered list and Unordered list tag

    Ordered list can create numeric value and Unorderd list can create bold. In this tag we can change this type just write with TYPE attribute and change the funcnality.
